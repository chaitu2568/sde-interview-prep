# SDE Interview Prep Tracker

Private repository for tracking Senior SDE interview preparation progress.

## Structure
```
STUDY_PLAN.md          # Complete 12-week day-by-day plan
tracking/
  progress.json        # Machine-readable progress data
  weekly-checkins/     # Weekly reflection notes
solutions/             # LeetCode solutions organized by pattern
  01-arrays-hashing/
  02-two-pointers/
  ... (16 pattern folders)
system-design/
  fundamentals/        # DDIA notes, concept summaries
  designs/             # Full system design write-ups
  mock-notes/          # Mock interview feedback
behavioral/            # STAR stories
website/               # GitHub Pages progress dashboard
  index.html           # Visual progress tracker
```
